HouseCare API Documentation

GET /bookings

Get all future uncancelled bookings

Since the user may have many bookings

Parameters:

  • page (optional) - The page to read
  • limit (optional) -

Returns:

  • total_count - The total number of bookings made by the User
  • count - The number of items in the current bookings array
  • bookings Array
    • booking_id - The booking ID
    • property_id - The property ID
    • cleaner_id - The cleaner ID
    • start - Start time of the booking

Example:

$ curl -X GET --header 'Accept: application/json' 'https://housecare.tokyo/api/bookings?api_token=7VLnHV9Pg6S5xA9-9_a8'

{
  "total_count": 202,
  "count": 20,
  "bookings":
    {
      "booking_id": 11269,
      "property_id": 1645,
      "cleaner_id": null,
      "start": "2017-05-08T11:00:00.000+09:00"
    }
}

GET /bookings/{booking_id}

Load details about the booking

Parameters:

  • booking_id - The ID of the booking to be examined

Returns:

  • id - ID of the booking

Example:

$ curl -X GET --header 'Accept: application/json' 'https://housecare.tokyo/api/bookings/17112?api_token=7VLnHV9Pg6S5xA9-9_a8'

{
    "id": 17112
}

Parameters:

  • property_id - The property ID to make the booking on

Returns:

HouseCare API Documentation

GET /properties

Get all unclosed properties

Since the user may have many properties, not all the properties will be returned in one API call. It is possible to see other properties by providing a page parameter

Parameters:

  • page (optional) - The page to read
  • limit (optional) -

Returns:

  • property_id

GET /properties/{id}

Load details about the booking

Parameters:

  • booking_id - The ID of the booking to be examined

Returns:

  • id - ID of the booking

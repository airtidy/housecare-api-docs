POST /properties/create_airbnb

Create Airbnb-type property

Create an Airbnb-type (Minpaku type) property. Airbnb-type properties are defined by the need for additional information (in addition to address) for cleaning, refills and coin laundry.

Note that there are several things required before a Property can have bookings: enter_method, guide_url, garbage must be set and an Estimate must be created on the property. These parameters do not have to be present when creating a property, but they must be set before a booking can be made.

Parameters:

  • postal_code - The postal code of the property. Must be formatted 'xxx-xxxx' with no 〒 sign.
  • address1 - The section of the address up to chome e.g.: "東京都渋谷区恵比寿三丁目" (please do not enter the remaining address as it is displayed to all cleaners)
  • address2 - The remaining part of the address. (This part is only displayed to cleaners who have taken the job.)
  • checkout_time - Checkout time. This must be formatted xx:xx in 24-hour time format. e.g.: "15:00" for 3 pm
  • checkin_time - Checkout time. This must be formatted xx:xx in 24-hour time format. e.g.: "11:00" for 11 am
  • listing_url - The URL to the Airbnb/Homeaway or any other short-term rental listing site.
  • enter_method (optional but required for booking) - An explanation on how to enter the property.
  • guide_url (optional but required for booking) - A URL to the customer guide to the property.
  • garbage (optional but required for booking) - An explanation of how to dispose of garbage.
  • nickname (optional) - A nickname for the property.

Returns:

  • property_id - The ID of the property

Example:

POST /properties/edit_airbnb/{property_id}

Edit Airbnb-type property

Modify or update the information of an Airbnb-type property. Simply pass in only the parameters you wish to modify.

Parameters:

  • postal_code - (optional) The postal code of the property. Must be formatted 'xxx-xxxx' with no 〒 sign.
  • address1 - (optional) The section of the address up to chome e.g.: "東京都渋谷区恵比寿三丁目" (please do not enter the remaining address as it is displayed to all cleaners)
  • address2 - (optional) The remaining part of the address. (This part is only displayed to cleaners who have taken the job.)
  • checkout_time - (optional) Checkout time. This must be formatted xx:xx in 24-hour time format. e.g.: "15:00" for 3 pm
  • checkin_time - (optional) Checkout time. This must be formatted xx:xx in 24-hour time format. e.g.: "11:00" for 11 am
  • listing_url - (optional) The URL to the Airbnb/Homeaway or any other short-term rental listing site.
  • enter_method (optional) - An explanation on how to enter the property.
  • guide_url (optional) - A URL to the customer guide to the property.
  • garbage (optional) - An explanation of how to dispose of garbage.
  • nickname (optional) - A nickname for the property.

Returns:

  • property_id - The ID of the property

Example:

$ curl -X POST --header 'Accept: application/json' -d 'postal_code=192-0782' 'https://housecare.tokyo/api/properties/edit_airbnb?api_token=7VLnHV9Pg6S5xA9-9_a8'

}

GET /properties

Get all unclosed properties

Since the user may have many properties, not all the properties will be returned in one API call. It is possible to see other properties by providing a page parameter

Parameters:

  • page (optional) - The (zero based) page number to read (Default 0)
  • limit (optional) - The number of properties to return (Default 20)

Returns:

  • total_count - Total number of properties
  • count - Number of properties returned
  • properties Array
    • property_id - ID of the property
    • address1 - Area address of the property
    • address2 - Building and room address of the property
    • nickname - Nickname of the property
    • has_ical - true if this property has iCals provided for automatic booking

Example:

$ curl -X GET --header 'Accept: application/json' 'https://housecare.tokyo/api/properties?api_token=7VLnHV9Pg6S5xA9-9_a8'

GET /properties/{property_id}

Load details about the property

Parameters:

  • property_id - The ID of the property to be examined

Returns:

  • property_id - ID of the property
  • address1 - Area address of the property
  • address2 - Building and room address of the property
  • nickname - Nickname of the property
  • has_ical - true if this property has iCals
  • guide_url - The guide URL of the property
  • listing_url - The listing URL of the property
  • garbage - The garbage instructions of the property
  • enter_method - Instructions on how to enter the property
  • booking_count - Number of future bookings for this property
  • default_checkout_time - The default check-out time for this property
  • default_checkin_time - The default check-in time for this property
  • default_duration - The default service time for this property
  • default_prev_no_of_guests - The default number of guests to clean up after
  • default_next_no_of_guests - The default number of guests to make the room for
  • ical_links Array - An array of iCal URLs provided for automatic booking

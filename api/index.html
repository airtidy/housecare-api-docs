HouseCare API endpoint

https://housecare.tokyo/api

It is highly recommended that the HTTPS endpoint for the HouseCare API be used because your sensitive information will be transmitted via plaintext otherwise, allowing wiretappers and other third parties to read this information.

In each of the API methods we provide an curl commandline call to the API to demonstrate concisely all the required information for making an API call successfully.

Authentication flow

Authentication on the HouseCare API is done by logging on and off using the sessions API. By logging on you will receive an API token that can be used to perform other functions on the website including logging off.

In all API calls except sessions, the api_token parameter is always required.

Example exchange:

John smith logs in

$ curl -X POST --header 'Content-Type: application/x-www-form-urlencoded' --header 'Accept: application/json' -d 'email=johnsmith%40housecare.tokyo&password=abcabcabcabc' 'https://housecare.tokyo/api/session'

{
    "api_token": "7VLnHV9Pg6S5xA9-9_a8"
}

Makes a booking

$ curl -X POST --header 'Content-Type: application/x-www-form-urlencoded' --header 'Accept: application/json' -d 'property_id=181&day=7&month=5&year=2017' 'https://housecare.tokyo/api/bookings?api_token=7VLnHV9Pg6S5xA9-9_a8'

{
    "booking_id": 2346,
    "surge_factor": "1.0"
}

Checks his new booking's information

$ curl -X GET --header 'Accept: application/json' 'https://housecare.tokyo/api/bookings/2346?api_token=7VLnHV9Pg6S5xA9-9_a8'

{
    "booking_id": 2346,
    "property_id": 181,
    "property_address1": "東京都大田区蒲田1-25-11",
    "property_address2": "アーユーキッディング 1M（ブラインド）",
    "postal_code": "160-0001",
    "nickname": "",
    "matched": false,
    "start": "2017-05-07T11:00:00.000+09:00",
    "checkout_time": "11:00",
    "checkin_time": null,
    "duration": 135,
    "prev_no_of_guests": 2,
    "next_no_of_guests": 3
}

And then logs out

$ curl -X DELETE --header 'Accept: application/json' 'https://housecare.tokyo/api/    bookings/2345?api_token=7VLnHV9Pg6S5xA9-9_a8'

{
    "booking_id": 2345,
    "cancelled": true
}

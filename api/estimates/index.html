GET /estimates/required_fields

Gets the required fields

Parameters:

  • language - Language used to display choices. Only en or ja supported

Returns:

  • Array
    • parameter - The name of the parameter required for estimation
    • description - A short human-readable description explaining the parameter
    • type - Either mulitplier or choice.
    • multiplier means the parameter accepts a number
    • choice means the parameter accepts one of the options defined in the options field
    • options - An array of the valid inputs for the parameter.

Example:

$ curl -X GET --header 'Accept: application/json' 'https://housecare.tokyo/api/estimates/required_fields?language=en&api_token=7VLnggggg6S5xA9-9_a8'

[
  {
    "parameter": "stairs",
    "description": "Information about stairs in the property",
    "type": "choice",
    "options": [
      "have_stairs",
      "no_stairs"
    ]
  },
  {
    "parameter": "living_room_count",
    "description": How many living rooms there are,
    "type": "multiplier"
  }
]

POST /estimates

Submit property information to retrieve a time estimate for cleaning

Parameters:

  • property_id - The ID of the property
  • other parameters - The names of these parameters can be found by calling GET /estimates/required_fields

Returns:

  • property_id - The ID of the property this estimate was created for
  • estimated_time_sec - The number of seconds this property's time is estimated for
  • created_at - The time when this estimate was created

Example:

$ curl -X POST --header 'Accept: application/json' -d 'property_id=1737&living_room_count=1&stairs=have_stairs' 'https://housecare.tokyo/api/estimates?api_token=7VLnggggg6S5xA9-9_a8'

{
  "property_id": 1737,
  "estimated_time_sec": 8100,
  "created_at": "2017-08-15T17:17:18.440+09:00"
}

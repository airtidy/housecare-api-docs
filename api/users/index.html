POST /users/sign_up

Create a user

Create a user with the provided information. The user is created first with no password, and when the user signs up the user is then sent an e-mail for the user to setup his password.

note on testing environments: Instead of sending an e-mail for the user a set password e-mail, testing environments will automatically set the password to placeholderpassword.

Parameters:

  • first_name - User's first name
  • last_name - User's last name
  • email - User's e-mail address. This e-mail address is where the setup password e-mail will be sent
  • phone - User's phone number. This is required for HouseCare to contact the user in case of trouble (keys to the room not found, an emergency happens in the room, etc...)

Returns:

  • user_id - The ID of the user

Example:

$ curl -X POST --header 'Accept: application/json' -d 'first_name=John&last_name=Smith&email=johnsmith%example.com&phone=0355512345' 'https://housecare.tokyo/api/users/sign_up'

{
  "user_id": 2586
}
